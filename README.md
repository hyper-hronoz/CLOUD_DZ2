# Опросник на Flask

Шуточный веб-опросник про музыкальные инструменты на картине Иеронима Босха
«Сад земных наслаждений».

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

После запуска откройте `http://127.0.0.1:5000/`.

## Что есть в проекте

- `app.py` - Flask-приложение с формой, подсчетом баллов и сохранением ответов.
- `data/responses.csv` - таблица с собранными ответами.
- `data/classmates.txt` - локальная копия списка одногруппников.
- `../dz1/students.json` - резервный источник, если локальный файл отсутствует.
- `analyze_results.py` - анализ результатов по требованиям задания.

## Анализ результатов

После сбора ответов выполните:

```bash
python3 analyze_results.py
```

Скрипт покажет:

- кто из списка `data/classmates.txt` не проходил опрос;
- какой вопрос был самым сложным;
- среднюю оценку;
- наиболее популярное время суток для прохождения.

## Важно

Скрипт `analyze_results.py` сначала использует `data/classmates.txt`.
Если локального файла нет, тогда берется `../dz1/students.json`.
